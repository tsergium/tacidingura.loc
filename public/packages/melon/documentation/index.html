Step1: Include Files

Include JavaScript and CSS files in HEAD tag.

                <link rel="stylesheet" type="text/css" href="timeline.css" />
                <script type="text/javascript" src="jquery.js"></script>
                <script type="text/javascript" src="timeline.min.js"></script>

Step2: Display Timeline

Add the HTML and Script below to the body of your HTML (where you want to display the timeline).

                <div id="timeline"></div>

                <script type="text/javascript">
                    $(document).ready(function() {
                        var timeline_data = [
                            {
                                type:        'blog_post',
                                date:        '2011-08-03',
                                dateFormat:  'DD MMMM YYYY',
                                title:       'Blog Post',
                                content:     '',
                                images:      ['xxxx.jpg'],
                                readmore:    'http://www.example.com'
                            },
                            {
                                type:        'blog_post',
                                date:        '2010-12-16',
                                dateFormat:  'DD MMMM YYYY',
                                height:      200,
                                images:      ['xxx.jpg', 'xxx.jpg'],
                                speed:       5000
                            }
                        ];

                        var timeline = new Timeline($('#timeline'), timeline_data);
                        timeline.display();
                    });
                </script>

Configuration options

setOptions(). For initial declarative timeline setup. Options are all optional.

                <script type="text/javascript">
                    var timeline = new Timeline($('#timeline'), timeline_data);
                    timeline.setOptions({
                        dateFormat:  'DD MMMM YYYY',
                        animation:   true,
                        lightbox:    true,
                        separator:   'year',
                        columnMode:  'dual',
                        order:       'desc',
                        max:         null
                    });
                    timeline.display();
                </script>
  • dateFormatString
    Default value: DD MMMM YYYY. Enable/Disable animation when timeline is first displayed. See http://momentjs.com/docs/#/displaying/format/ for more details.
     
    Month
    M1 2 ... 11 12M
    Mo1st 2nd ... 11th 12th
    MM01 02 ... 11 12
    MMMJan Feb ... Nov Dec
    MMMMJanuary February ... November December
     
    Day
    D1 2 ... 30 30
    Do1st 2nd ... 30th 31st
    DD01 02 ... 30 31
     
    Year
    YY70 71 ... 29 30
    YYYY1970 1971 ... 2029 2030
  • animationBoolean
    Default value: true. Enable/Disable animation when timeline is first displayed.
  • lightboxBoolean
    Default value: true. Enable/Disable lightbox feature when an image is clicked.
  • separatorString
    Default value: year. Set separator type. Possible values: null, year, month, month_year
  • first_separatorBoolean
    Default value: false. Set to display the first separator. Possible values: true, false
  • columnModeString
    Default value: dual. Set what column mode to use. Possible values: dual, left, right, center.
  • orderString
    Default value: desc. Set timeline elements order. Possible values: desc, asc.
  • maxInt
    Default value: null. Set maximum number of timeline elements to be displayed.
  • loadmoreInt
    Default value: 0. Set how many additional data to load when clicking on "Load More". This options needs to be used with max together.
  • facebookAccessTokenString
    Default value: null. Set the facebook access token required to access Facebook Posts. The access token can be found at "https://graph.facebook.com/oauth/access_token?client_id={app-id}&client_secret={app-secret}&grant_type=client_credentials".
  • facebookPageIdInt
    Default value: null. Set where to pull the facebook posts from.

Timeline Data

Timeline data is an array of JSON objects. Each object represents one timeline element. Different element type has different object properties.

                var timeline_data = [
                    {
                        type:        'blog_post',
                        date:        '2011-08-03',
                        dateFormat:  'DD MMMM YYYY',
                        title:       'Blog Post',
                        content:     '',
                        images:      ['xxxx.jpg'],
                        readmore:    'http://www.example.com'
                    },
                    {
                        type:        'blog_post',
                        date:        '2010-12-16',
                        dateFormat:  'DD MMMM YYYY',
                        images:      ['xxx.jpg', 'xxx.jpg'],
                        speed:       5000
                    }
                ];
  • blog_post
                                {
                                type:        'blog_post',
                                date:        '2011-08-03',
                                dateFormat:  'DD MMMM YYYY',
                                title:       'Blog Post',
                                content:     '',
                                image:       '001.jpg',
                                readmore:    'http://www.example.com'
                            }
  • gallery
                                {
                                type:        'gallery',
                                date:        '2011-08-03',
                                dateFormat:  'DD MMMM YYYY',
                                title:       'Blog Post',
                                image:        ['xxx.jpg', 'xxx.jpg']
                            }
  • iframe
                                {
                                type:        'iframe',
                                date:        '2011-08-03',
                                dateFormat:  'DD MMMM YYYY',
                                title:       'Blog Post',
                                height:      300,
                                url:         'http://www.youtube.com/embed/0ZQBRsEyN1E?wmode=opaque'
                            }
  • typeString
    Default value: blog_post. Define timeline element type. Possible values: blog_post, gallery, iframe.
  • dateString
    Default value: 2000-01-01. Set timeline element data. Format: yyyy-mm-dd. When month or day are unknown, set them to 01, then use dateFormat to not display them.
  • dateFormatString
    Default value: null (02 February 2013). Specify a dateFormat in element data overwrites the overall dateFormat option
  • heightInteger
    Default value: 300. Set element height for gallery or iframe.
  • titleString
    Default value: null. Set timeline element title.
  • contentString
    Default value: null. Set content for blog post or gallery element.
  • imageString
    Default value: null. Set image for blog post element.
  • readmoreString
    Default value: null. Set read more URL for blog post element.
  • imagesArray
    Default value: empty. An array of images used in gallery or blog post element.
  • speedInteger
    Default value: 5000. Set animation speed for sliders in blog post element in million seconds.
  • urlString
    Default value: null. Set iframe URL for iframe element.
